# Micro Journal App

## Concept
A micro journaling application that prompts users at regular intervals to create quick journal entries, then consolidates these entries into daily journal files compatible with various journaling platforms.

## Core Features

### Entry Creation
- **Regular Prompts**: Configurable intervals for journal entry prompts
- **Rich Content Support**:
  - Text entries
  - Photo attachments
  - Video attachments
  - Other file attachments
- **Quick Entry Interface**: Streamlined UI for fast input

### Daily Consolidation
- **End-of-day Processing**: Automatically consolidate micro entries into a single daily journal entry
- **Multi-format Export**: Support for various journaling platforms:
  - Obsidian (Markdown with metadata)
  - Apple Notes
  - Plain text files
  - Markdown files
  - Custom formats as needed

### Cross-Platform Support
- **Primary Targets**: macOS and Web (Phase 1)
- **Future Platforms**:
  - Mobile (iOS/Android)
  - Windows
  - Linux

## User Experience Goals
- Minimal friction for quick entries
- Seamless integration with existing journaling workflows
- Configurable prompting system
- Reliable data backup and sync

## Technical Requirements
- Cross-platform compatibility
- Local storage with sync capabilities
- Rich media handling
- Export format flexibility
- Notification system for prompts
